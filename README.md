### How do I get set up? ###

* intall python 2.7.13 if you don't have
* git clone 
* $ cd ClinicalTrails (This is git repo's root folder)
* $ virtualenv -p /usr/bin/python2.7 env
* $ source env/bin/activate
* $ pip install -r requirements.txt
* $ python runserver.py
